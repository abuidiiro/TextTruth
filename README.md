# TextTruth

**An explainable AI text detection system built with R Shiny**

## Live Demo
🔗 [Try TextTruth here](https://abuidiiro.shinyapps.io/hybrid-ai-detector/)

## Features

- **Explainable Results** – Shows why text is classified as AI or Human
- **3-Class System** – Human-Written | Uncertain | AI-Generated
- **Intellectual Depth Score** – 0-12 scale measuring original thinking
- **Honest Uncertainty** – Returns "Uncertain" for short texts
- **30+ Linguistic Features** – Burstiness, perplexity, and more

## Performance

| Metric | Value |
|--------|-------|
| Accuracy | 87.7% |
| Macro F1 | 0.878 |
| AUC | 0.970 |

## Depth Score Guide

| Score | Meaning |
|-------|---------|
| 0–3 | Strong AI indicators |
| 4–7 | Mixed signals - Manual review |
| 8–12 | Strong human characteristics |
